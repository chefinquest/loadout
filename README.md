# Loadout

A Call of Duty-inspired product recommendation screen that turns a mocked Twitter/X profile into a tactical product loadout.

## Features

- Enter a Twitter/X handle
- Mocked profile intelligence and vibe classification
- Product recommendation slots: Primary, Secondary, Perk, Tactical, Lethal, Field Upgrade
- COD/HUD-inspired interface
- Static export for GitHub Pages

## Mock handles

- `buildlord`
- `cybermuse`
- `trailstack`

Unknown handles fall back to a generic Mystery Operator profile.

## Local development

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:3000`.

## Verification

```bash
npm test
npm run lint
npm run build
```
